Cherry Blossom

"Embark on a fun-filled adventure in Cherry Blossom, a thrilling PyGame creation! As the player, you are tasked with catching the fruit as it falls and dodging the angry bees. Can you rack up the highest score before the clock runs out?"

github link

Project Overview

"In the course of a five-day team project, my partner and I were tasked with interpreting the prompt "Blossom". We decided to create a game that would be fun for all ages. We used PyGame to create the game, and Python to create the logic. The game is a simple arcade-style game where the player must catch the falling fruit and avoid the angry bees. The player has 60 seconds to rack up the highest score possible. The game is challenging, but fun! We hope you enjoy it!"
